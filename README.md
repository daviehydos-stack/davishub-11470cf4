# Azani ISP Project

A complete Internet Service Provider (ISP) project management system for KCSE 2026 Computer Studies.

## Features

- Complete documentation and database
- Milestone 1 and Milestone 2 packages
- Free guides and resources
- WhatsApp support

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- Supabase (Database + Auth)
- Framer Motion

## Development

```bash
npm install
npm run dev
```

## Deployment

The project is deployed automatically via Vercel.
